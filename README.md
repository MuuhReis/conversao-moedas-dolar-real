# conversao-moedas-dolar-real
Projeto desenvolvido em Java que realiza a conversão de moedas entre Dólar (USD) e Real (BRL), com interação via terminal.

⸻

Funcionalidades
• Menu interativo com opções:
  • Converter Dólar → Real 
  • Converter Real → Dólar 
  • Sair do programa
  • Entrada de valores via teclado
  • Inserção manual da cotação da moeda
  • Cálculo automático da conversão
  • Validação de valores inválidos

⸻

Tecnologias utilizadas
  • Java
  • Classe Scanner para entrada de dados
  • Estrutura condicional (switch-case)
  • Estrutura de repetição (while)
  • Métodos para organização do código

⸻

Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:
  • Lógica de programação
  • Estruturas de decisão (switch-case)
  • Uso de métodos para modularização
  • Validação de entradas do usuário
  • Interação com o usuário via terminal

⸻

Exemplo de execução

O usuário escolhe o tipo de conversão, informa o valor e a cotação da moeda, e o programa retorna o resultado convertido no terminal.
